## 版本说明

- **v0.5**（当前）：Message 系统重构（双消息列表）、CLI 线程化、IME 兼容
- **v0.1 Demo**：初始版本，单消息列表架构

历史存档中的文档为 v0.1 时代的原始设计，已过时但保留作为参考。

## 当前实现状态

- CLI 是当前可运行主线：`main.py` 维护 `ScenesoulLoop`、后台任务队列、双消息列表和状态拼接。
- Web UI 代码仍在 `ui/web_server.py`，但尚未迁移到 v0.5 Agent 接口；涉及 Web 的文档按“待迁移/旧接口”理解。
- 驱动力由 `profiles/<name>/soul.md` 的 `traits` 初始化，并由界说通过 `update_drives` 工具显式更新；当前没有每轮自动衰减函数。
