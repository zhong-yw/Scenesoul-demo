## 一、技术栈

| 层级 | 技术 | 版本 | 用途 |
|------|------|------|------|
| 语言 | Python | 3.10+ | 主开发语言 |
| LLM SDK | openai | ≥1.0 | OpenAI 兼容 API 调用 |
| Web 框架 | Flask | ≥2.0 | Web UI 后端 |
| 环境变量 | python-dotenv | ≥1.0 | `.env` 文件加载 |
| 配置 | PyYAML | ≥6.0 | Profile YAML frontmatter 解析 |
| 终端输入 | Windows API (`ctypes`) | — | `ReadConsoleInputW()`（IME 兼容） |
| 线程 | `threading` + `queue` (内置) | — | 后台 LLM 调用 |

使用 OpenAI 兼容 SDK，可对接任何兼容 OpenAI API 格式的服务（DeepSeek、Ollama、MiniMax 等）。

## 三、环境变量配置 (.env)

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `OPENAI_API_KEY` | （必填） | LLM API Key |
| `OPENAI_BASE_URL` | `http://127.0.0.1:8080/v1` | API 端点 |
| `OPENAI_MODEL` | `ds-flash` | 模型名 |
| `THINK_INTERVAL` | `10` | 大脑思考间隔（秒） |
| `USER_TIMEOUT` | `600` | 用户超时判定（秒，10 分钟） |

## 四、启动方式

### 4.1 CLI 模式

```bash
python main.py                     # 默认启动（使用 default profile）
python main.py --preset bedroom_warm  # 使用指定预设
python main.py --list-presets      # 列出所有可用预设
python main.py --debug             # 显示界说原始 LLM 输出
```

CLI 模式依赖 Windows API（`ReadConsoleInputW`），支持拼音等 IME 输入。

### 4.2 Web UI 模式

Flask 后端提供三个 REST API + 前端页面，详见 [ui/web_server.py](ui/web_server.py)。

## 五、配置文件

### settings.json

[world/settings.json](world/settings.json) 包含世界的基础设定（时段描述、人格参数）。

### Profile 预设

每个预设是 `profiles/<名称>/` 目录下的 6 个 Markdown 文件：

| 文件 | 用途 |
|------|------|
| `soul.md` | 人格核心描述 |
| `brain.md` | 大脑行为指令 |
| `memory.md` | 初始记忆设定 |
| `narrator.md` | 界说行为指令 |
| `world.md` | 世界的模样 |
| `scene.md` | 场景定义 |
