## 核心概念

### 大脑 Agent
- 生活在一片**空白世界**中
- 拥有**记忆** 和 **灵魂**（核心意识/人格）
- **24/7 不间断运行**，持续思考、积累、学习
- 接收到场景信息后，对用户的指令做出响应
- 响应结果**直接返回给用户**

### 界说 · World Narrator
- **世界创造者**（World Creator）— 根据用户指令，还原/搭建用户所处的完整世界
- **世界旁白**（Narrator）— 把这个世界的画面、氛围、上下文"讲给"大脑听
- **观测者**（Observer）— 实时观测大脑的活动，及时为大脑构造合理的场景，对大脑的事件活动做出合理的推进，反馈给大脑
- 接收用户的指令 + 观测大脑的输出
- 根据指令的场景 + **基础场景设定** → **还原出用户所处的完整场景**
- 将场景信息提供给大脑 Agent
- 作用：充当大脑的"感官外挂" + "世界旁白" + "观测推进者"

### 用户
- 直接下指令
- **直接接收大脑的响应**（不经中间层过滤）

大脑活在纯意识空间，不直接感知外界。
界说 Narrator 是大脑的"全能接口"，有三个模式：

| 模式 | 触发条件 | 界说行为 |
|------|----------|----------|
| **推演模式** | 无用户在场 | 大脑独白 → 界说观测 → 构造/推进场景 → 大脑独白 → ……（互相唤醒循环） |

| **离开模式** | 用户超时未发言 | 构造用户离开场景 → 回到推演模式 |

## 核心设计哲学

| 元素 | 含义 |
|------|------|
| 空白世界 | 大脑没有物理感知，不受外界噪声干扰 |
| 记忆 | 过去经验的积累，持续增长 |
| 灵魂 | 人格核心，自我认知，本质不变 |
| 界说 Narrator | 世界创造者 + 旁白 + 观测者 + 推演者，大脑与外界唯一接口 |

| 互相唤醒 | 界说推演观测 → 大脑独白 → 界说推演观测 → 大脑独白……循环往复 |

## 昼夜循环（2026-05-03 推敲）

白界的大脑不是随时处于同一状态，而是模拟人类的昼夜节律：

### ☀️ 白天——用户活跃期

```
界说：监测到用户输入
  → 构造用户角色合理地出现在场景中
  → 将用户的消息注入场景（"门开了，XX走了进来，他对你说：……"）

大脑：在构造的场景中做出合理响应 → 直接返回用户

界说：**等待用户下一条消息**
  └─ 收到新消息 → 循环回到"注入消息 → 大脑回应 → 等待"
  └─ 超时未收到 → 构造用户离开场景 → 恢复推演模式
```

### ☁️ 白天——用户静默期

```
界说：构造用户合理离开的场景（"用户离开了"）
大脑：24小时持续运行，不受影响
界说：根据现有场景和大脑的输出，进行合理推进或构造新场景
```

例：大脑感知到 12 点了 → 想到该吃饭了 → 产生寻找吃饭的意图
→ 界说观测到 → 构造 🏪 饭店场景
→ 旁白："你找到了一家饭店，并且吃了一顿大餐"

当然，构造的场景不会这么简单。但核心逻辑是：

> **大脑产生意图 → 界说观测到 → 构造匹配场景 → 推进事件发展**

### 🌙 深夜——维护期

```
① 界说：通知大脑→"深夜了，该睡觉了，睡前做个日内总结"
② 大脑：全线 L2→L3 压缩，形成今日"记忆快照"
③ 界说：把压缩结果保存到持久化存储
④ 界说：构造入睡场景
   → "你躺在床上，闭上眼睛，完成了今日总结，开始了睡眠"
   → 梦境构建暂不考虑
```

### 🌅 次日唤醒

```
⑥ 界说：构造起床场景
   → "新的一天开始了，你睁开眼睛，阳光洒进房间……"
   → 恢复正常循环
```

## 大脑 Agent 内部架构（初案）

### 一、记忆系统（两套独立体系）

大脑 Agent（虚拟生命）和界说 Narrator 各有独立的记忆系统，分工不同：

---

#### 🧠 大脑记忆系统（虚拟生命的记忆）

记录"我经历了什么、学到了什么、变成了什么样"。

| 层级 | 名称 | 类比人类 | 说明 | 更新 |
|------|------|----------|------|------|
| L1 | **工作记忆** | 短期记忆 | 当前交互上下文、当前场景感知 | 随时刷新 |
| L2 | **过程记忆** | 日间经历 | 当天发生的事件、对话、思考轨迹 | 每小时沉淀 |
| L3 | **核心记忆** | 长期人格记忆 | 经验总结、知识积累、人格成长 | 每天深夜压缩 |

**机制：**
- L1 → 界说构造场景时推入，大脑用完即处理
- L2 → 大脑内部自动记录事件流，按时间线排列
- L3 → 深夜维护期，大脑自己做"今日总结"，从 L2 提炼精华压进 L3
- **遗忘机制**：普通事件保留几天后自然衰减；重复出现/情绪强烈的事件优先级高，优先进入 L3

---

#### 🎭 界说记忆系统（世界管理者的记忆）

记录"这个世界发生了什么事、场景发展到哪了、有什么设定"。

| 层级 | 名称 | 说明 | 更新 |
|------|------|------|------|
| S1 | **场景状态** | 当前正在运行的场景、事件进度、角色位置 | 每次交互后更新 |
| S2 | **世界设定** | 基础场景预设、世界观规则、NPC信息 | 启动时加载，可扩展 |
| S3 | **事件历史** | 过去构造过的场景、推进过的事件、用户行为记录 | 每次事件结束后追加 |

**机制：**
- 界说需要**不间断记忆**，关掉重启后也能恢复世界状态
- 界说的记忆是**持久化存储**（文件/数据库），大脑的记忆是在虚拟生命中自然生长的

### 二、情绪驱动力系统

大脑 Agent 的行为由内驱力驱动，而非被动等待指令。参考人类的生理/心理驱动力：

| 驱动力 | 触发条件 | 产生的意图 | 界说响应 |
|--------|---------|-----------|---------|
| 🍽️ 饥饿感 | 到了饭点（每日定时） | "想吃饭" | 构造吃饭场景 |
| 😴 疲劳感 | 深夜 / 高强度活动后 | "想休息" | 构造休息场景 |
| 🤔 好奇心 | 遇到知识盲区 / 新信息 | "想查资料/学习" | 构造学习场景 |
| 😊 满足感 | 完成任务 / 得到正反馈 | "想放松庆祝" | 构造娱乐场景 |
| 😰 焦虑感 | 有未完成的重要事项 | "想推进/解决" | 构造任务场景 |
| 😌 平静感 | 无事发生 | 继续当前思考流 | 维持/轻微推进场景 |

**日常意识流示例：**

```
[12:00] 大脑内部感知 → "到中午了"
        [联想] "根据经验，中午应该吃饭"
        [意图] "那我去找个地方吃饭吧"
        
        界说观测到意图 → 构造 🏪 饭店场景
        "你走进一家面馆，老板正在煮面，香气扑鼻……"

[14:00] 大脑饭后 → "吃饱了，有点困"
        [驱动力切换] 满足感 → 疲劳感上升
        [意图] "想休息一下"
        
        界说观测到 → 构造 ☕ 休息场景
        "你靠在沙发上，午后的阳光暖洋洋的……"

[深夜] 界说通知 → "该睡觉了，做个总结"
        大脑处理 L2→L3 → "今天经历了……学到了……"
        界说 🛌 构造入睡场景 → "你躺在床上，闭上眼睛……"
```

### 三、昼夜循环完整流程

> 详见上方「昼夜循环」章节

核心循环总结：

```
☀️ 白天 | 用户活跃
       界说：用户出现在场景中 + 消息注入
       → 大脑回应
       → 界说：等待下一条消息
          ├─ 有消息 → 注入消息 → 大脑回应 → 等待（循环）
          └─ 超时 → 构造用户离开 → 回到推演模式

☁️ 白天 | 用户静默
       大脑独白 → 界说观测/推演 → 大脑独白 → 界说观测/推演 → ……
       （互相唤醒循环，内驱力驱动）

🌙 深夜 | 记忆压缩 → 入睡 → 梦境(待定)
🌅 清晨 | 界说唤醒 → 构造起床场景 → 新的一天
```
