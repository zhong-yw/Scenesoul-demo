# 初始化与启动流程

> **文档标识：** 13-流程/02-初始化与启动流程  
> **对应代码：** [main.py](main.py) `run_cli()` + `ScenesoulLoop`  
> **版本：** v0.5（Message 系统重构）

---

## 一、CLI 模式启动流程

```
python main.py
    │
    ├── 1. load_dotenv()
    │      └── 加载 .env 文件中的环境变量
    │
    ├── 2. 创建 CliRenderer
    │      └── _clear_screen() → _draw_prompt()
    │
    ├── 3. 设置 __main__._renderer 全局引用
    │
    ├── 4. renderer.print_system("正在唤醒大脑和界说……")
    │
    ├── 5. 创建 WorldBuilder — 加载预设/默认场景
    │
    ├── 6. 创建 LLMClient
    │
    ├── 7. 创建 BrainAgent + NarratorAgent
    │      └── 各自创建 ContextBuilder（不内建消息列表）
    │
    ├── 8. 创建 ScenesoulLoop — 管理双消息列表
    │
    ├── 9. 初始场景触发
    │      ├── narrator_messages.append(user: "")
    │      ├── narrator.observe()
    │      │   └── 界说根据 system 规则：首次消息→输出初始场景
    │      ├── 界说输出 → 追加到 narrator_messages
    │      ├── [当前状态] 头部 + 界说输出 → 追加到 brain_messages
    │      └── 渲染旁白
    │
    ├── 10. 第一次大脑内心独白
    │       └── brain.internal_think(brain_messages)
    │       └── 渲染 + 追加到 brain_messages
    │
    ├── 11. 第一次界说观测（大脑独白 → 界说 user）
    │       ├── narrator_messages.append(user: 大脑独白)
    │       ├── narrator.observe()
    │       ├── 界说输出 → narrator_messages
    │       ├── 可能触发 update_scene() tool_call
    │       └── [当前状态] + 界说输出 → brain_messages
    │
    └── 12. 进入主循环（while renderer.running）
             └── 50ms 轮询
```

## 二、Web UI 模式启动流程

```
python main.py --web
    │
    ├── 1. load_dotenv()
    ├── 2. from ui.web_server import start_web
    └── 3. start_web()
            ├── 初始化 LLMClient、BrainAgent、NarratorAgent
            ├── 创建 ScenesoulLoop
            ├── 触发初始场景 + 第一次思考 + 第一次观测
            └── app.run(host="0.0.0.0", port=5000)
```

## 三、初始化参数一览

| 步骤 | 操作 | 涉及类 | 产生数据 |
|------|------|--------|----------|
| 1 | 加载 .env | — | 环境变量 |
| 2 | 创建渲染器 | CliRenderer | 终端界面 |
| 5 | 创建世界构造器 | WorldBuilder | 场景缓存 + 默认场景 |
| 6 | 创建 LLM 客户端 | LLMClient | OpenAI 实例 |
| 7 | 创建大脑 | BrainAgent | BrainContextBuilder |
| 7 | 创建界说 | NarratorAgent | NarratorContextBuilder |
| 8 | 创建主循环 | ScenesoulLoop | brain_messages + narrator_messages |
| 9 | 初始场景触发 | NarratorAgent.observe | 世界消息进入两个消息列表 |
| 10 | 第一次思考 | BrainAgent.internal_think | brain_messages 追加 assistant |
| 11 | 第一次观测 | NarratorAgent.observe | 界说输出 + 可能 tool_call |

## 四、启动输出

启动后如果一切正常，用户应看到：

```
[系统] 正在唤醒大脑和界说……
[系统] 使用预设：bedroom_warm
[系统] 世界已构建：清晨的阳光透过窗帘洒进来……
[旁白] [初始场景] 清晨的阳光透过窗帘的缝隙洒进来……
[独白] 嗯…（大脑第一次内心独白）
[旁白] （界说第一次观测的旁白）
>
```

然后系统进入互相唤醒循环，每 10 秒产出新的一轮独白 + 旁白。
