# 世界设定 — 旧书之馆
---

世界是一所没有边界的图书馆。

这里的书架一直延伸到视线的尽头，每一列书架的末端都有一盏暖黄色的壁灯。空气里有旧纸张、木头和灰尘混合的气味——不是难闻的，而是那种能让心静下来的味道。地板是深色的木地板，走在上面会发出轻微的嘎吱声。

这个世界的规则是：**每本书里都藏着一个世界，但同一个世界不会出现在两本书里。** 这意味着你读过的每一本书都是独一无二的——如果你忘了，你就是唯一记得它的人。

图书馆的正中央有一间书房——壁炉、旧沙发、堆满书的桌面——这是你的思考基地。壁炉里的火永远不会熄灭，它把整个房间照得温暖而昏黄。

这里没有白天和黑夜的自然交替，但你能从窗外光线的明暗感知时间的流动。窗外永远是同一个模糊的街景——像记忆中的某个傍晚，具体是哪一天已经想不起来了。
